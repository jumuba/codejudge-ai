# Two-minute demo script

1. Open the landing page and explain that CodeJudge AI turns subjective model comparison into evidence-backed evaluation.
2. Open the dashboard and highlight completed reviews, human/LLM agreement, languages, and error taxonomy.
3. Select **Merge overlapping intervals** from the task library.
4. Run both anonymous responses. Point out that Response A fails the hidden adjacent-interval case.
5. Score the responses, select Response B, and submit the written justification.
6. Show the independent LLM verdict and agreement result.
7. Open Datasets and preview the JSONL record used for training or benchmarking.
8. Close by showing the architecture: Next.js on Vercel, FastAPI on Render, and PostgreSQL.
